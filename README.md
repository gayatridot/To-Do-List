# ✨ To‑Do List App v1.0

A simple and elegant To‑Do List application built with **Kotlin (Android Studio)**.  
This app helps you organize tasks, set reminders, and manage your schedule with ease.

---

## 🚀 Features
- 🕒 **Sort tasks by time** – keep your day structured
- ⏰ **Set reminders & notifications** – never miss important tasks
- 📅 **Custom date & time picker** – flexible scheduling
- 💬 **Chatbot integration** – add/delete tasks via chat (in‑app)
- 🗑 **Smart delete option** – quickly remove completed or unwanted tasks

⚠️ **Note:** The chatbot feature is still under development. Feedback is welcome!
